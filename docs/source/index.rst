JJiGiT Documentation
====================

Welcome to JJiGiT's official documentation!

**JJiGiT** is an open-source, real-time polling platform designed to support transparent and smart decision-making.
It provides all the tools you need whether you are organizing team votes, conducting surveys, or facilitating structured discussions.

Quick Links
-----------

* `GitHub Repository <https://github.com/OSS-Group11/jjigit-be>`_
* `Issue Tracker <https://github.com/OSS-Group11/jjigit-be/issues>`_
* `Contributing Guide <contributing.html>`_

Features
--------

* **Real-Time Voting** - See results update live as votes come in.
* **Smart Visualizations** - Interactive charts for better insights.
* **Public Polls** - Control who can access your polls.
* **Transparency** - Comments show what the author voted for.
* **Open Source** - Free, transparent, and community-driven.
