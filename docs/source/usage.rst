How to Use
==========

This guide covers the core features of JJiGiT: creating polls, voting, viewing results, and commenting.

Creating a Poll
---------------

1. **Log In**: You must be logged in to create a poll.
2. **Click Create**: Click the "Create Poll" button on the main dashboard.
3. **Enter Details**:
   * **Title**: Enter the question or topic.
   * **Options**: Add at least 2 options (e.g., "Yes", "No").
   * **Visibility**:
     * ``Public``: Visible on the main list, open to everyone.
     * ``Private``: Accessible only via direct link.
4. **Submit**: Click create to finish. You will be redirected to the poll page.

.. note::
   Once created, polls cannot be edited to ensure integrity. Please check for typos before submitting.

Voting
------

1. **Select Poll**: Click on a poll from the list or open a shared link.
2. **Choose Option**: Click on the option you support.
3. **Submit Vote**: Click the "Vote" button.

.. warning::
   **One Person, One Vote**: You can only vote once per poll. Votes cannot be changed or cancelled after submission.

Viewing Results
---------------

JJiGiT provides real-time results.

* **Instant Updates**: You can see results immediately without voting.
* **Real-Time Graphs**: Charts update instantly when other users vote.
* **Statistics**: View exact vote counts and percentages for each option.

Commenting
----------

You can discuss the poll topic with other users.

1. Scroll down to the comment section on the poll detail page.
2. Type your opinion in the text box.
3. Click "Post Comment".

**Transparency Badge**:
When you post a comment, an icon showing **which option you voted for** will be displayed next to your username to provide context.
