About the Project
==================

What is JJiGiT?
---------------

JJiGiT is an open-source, real-time polling platform designed to support transparent and smart decision-making.
It provides all the tools you need whether you are organizing team votes, conducting surveys, or facilitating structured discussions.
The platform is completely free, ad-free, and comes with no usage restrictions.

Project Purpose
---------------

JJiGiT aims to create a space where people can:

* Organize quick, informal polls for fun debates (e.g., "Is a straw hole one or two?")
* Conduct serious decision-making polls for teams and communities
* Visualize voting results in real-time with interactive charts
* Facilitate structured discussions through comments and feedback

The platform emphasizes transparency and community-driven development, making all code and features fully open source.

Key Features
------------

* **Create & Share Polls** - Generate unique links to share your polls instantly
* **Real-Time Voting** - See results update live as votes come in
* **Smart Visualizations** - Interactive charts powered by Chart.js
* **Topic Suggestions** - Get automated topic generation assistance
* **One Vote Per User** - Fair voting system with strict 1-person-1-vote policy
* **Public & Private Polls** - Control visibility with public or link-only access
* **Comment System** - Enable discussions on each poll with comment threads

Technology Stack
----------------

The project consists of three main repositories:

Backend
~~~~~~~

* **Framework**: Spring Boot 3.4.2
* **Language**: Java 21
* **Database**: MySQL 8.0+
* **ORM**: Spring Data JPA
* **Build Tool**: Gradle
* **API Documentation**: Swagger/OpenAPI

Frontend
~~~~~~~~

* **Library**: React 18+
* **Language**: JavaScript
* **Styling**: Tailwind CSS
* **Charts**: Chart.js
* **HTTP Client**: Axios
* **State Management**: React Hooks

Machine Learning
~~~~~~~~~~~~~~~~

* **Language**: Python 3.9+
* **Framework**: Flask / FastAPI
* **ML Library**: Hugging Face Transformers
* **Models**: LLaMA-based topic generation
* **Tasks**: Topic Generation, Text Summarization
